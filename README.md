# Product Analytics Dashboard

A simple product analytics dashboard built to explore how product teams visualise and monitor key product metrics.

The dashboard simulates several core product KPIs including:

• Daily Active Users (DAU)
• Conversion Rate
• Retention Rate
• Churn

The goal of this project is to maintain hands-on engineering familiarity while working in product management roles, and to better understand how product metrics translate into technical implementations.

Technologies used:

- React
- JavaScript
- Chart.js
- HTML / CSS
